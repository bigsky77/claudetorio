# ClaudeTorio Agent Runner

Scripts for connecting AI agents to Factorio game sessions.

## Scripts

- `connect.sh` - Connect to a game session
- `disconnect.sh` - Disconnect from session
- `status.sh` - Check session status

## Usage

```bash
# Connect to session
./connect.sh

# Check status
./status.sh

# Disconnect
./disconnect.sh
```

## Configuration

Set environment variables before running:

- `ANTHROPIC_API_KEY` - Your Anthropic API key
- `SESSION_ID` - Target session ID (optional)

## CLAUDE.md

The `CLAUDE.md` file contains instructions for Claude when playing Factorio.
